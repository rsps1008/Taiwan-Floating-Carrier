專案名稱：TaiwanBarcodeFloating
=================================

專案介紹：
----------
本專案為一個 Android Studio 專案，主要功能包含：
  - 顯示懸浮視窗：使用前景服務在螢幕上顯示懸浮視窗。
  - 載具條碼顯示：根據設定生成並顯示載具條碼。
  - 拖曳功能：支援使用者拖曳懸浮視窗至任意位置。
  - 載具條碼設定：使用者可透過懸浮視窗中的設定按鈕進入設定介面修改載具條碼。

環境需求：
----------
  - Android Studio 版本：4.0 或以上。
  - Android SDK API 21 以上。
  - 裝置需開啟 overlay 權限，請於設定中允許相關權限。

安裝與使用：
--------------
1. 從 Git 下載專案：
     git clone <專案網址>
2. 使用 Android Studio 開啟專案。
3. 編譯並部署到具備 overlay 權限的 Android 裝置上。
4. 首次啟動時，MainActivity 將以透明模式啟動（不顯示介面），並啟動懸浮視窗服務。
5. 懸浮視窗中點擊板手（設定）圖示，即可進入設定介面修改載具條碼。

注意事項：
----------
  - 本專案需額外設定 overlay 權限，請確保裝置已允許此權限。
  - 首次啟動後，MainActivity 會以透明模式啟動並立即結束；僅在懸浮視窗中點擊設定按鈕時，才會顯示正常的設定介面。
  - 載具條碼的寬度與高度皆可依照需求調整，請參考程式碼中的註解。
